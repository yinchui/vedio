# UI设计提示词 - 供Pencil使用

**项目名称：** AI智能剪辑助手
**日期：** 2026-02-14
**用途：** 在Pencil中预览和设计UI界面

---

## Pencil UI设计提示词

```
请设计一个专业的桌面视频剪辑软件界面，类似剪映或Premiere Pro的布局风格。

### 整体布局要求：

1. **顶部区域（高度约80px）**
   - 第一行：菜单栏（文件、编辑、视图、工具、帮助）
   - 第二行：工具栏
     - 左侧按钮：[导入素材] [AI自动分析] [导出]
     - 右侧按钮：[撤销] [重做] [设置]

2. **中间工作区域（分为三栏）**
   - **左侧栏（宽度20%）：素材库区域**
     - Tab切换：「素材列表」「AI分析结果」
     - 素材列表：网格布局显示视频卡片
       - 每个卡片：缩略图 + 文件名 + 时长标签（如"02:34"）
     - AI分析结果：列表显示分析片段
       - 每项格式：
         ```
         ✅ 片段1 (00:10-00:35)
         「海边日落，画面优美」
         精彩度：92分
         [建议保留]
         ```

   - **中间栏（宽度50%）：视频预览窗口**
     - 上方：视频播放器（16:9比例的黑色播放区域）
     - 下方：播放控制条
       - 左侧：[后退] [播放/暂停] [前进]
       - 中间：进度条 + 时间显示（00:00:30 / 00:03:45）
       - 右侧：[音量控制] [全屏]

   - **右侧栏（宽度30%）：属性面板**
     - 标题：根据选中元素动态显示（视频属性/字幕编辑/音频设置）
     - 内容区：表单控件
       - 示例（视频属性）：
         - 时长：00:00:25
         - 入点/出点时间输入框
         - 音量滑块
         - 变速单选按钮（正常/慢动作/快动作）
       - 示例（字幕编辑）：
         - 文本输入框
         - 字体选择下拉框
         - 字号滑块
         - 颜色选择器
         - 动画效果下拉框

3. **底部区域（高度30%）：时间轴编辑器**
   - 顶部：时间刻度尺
     - 显示时间标记：0:00, 0:30, 1:00, 1:30...
     - 红色播放指针（竖线）
   - 多轨道布局（从上到下）：
     - **视频轨道**：显示为蓝色矩形块，内含缩略图条带
     - **音频轨道**：显示为绿色波形
     - **字幕轨道**：显示为黄色文本块
     - **配乐轨道**：显示为橙色音乐波形
   - 每个轨道左侧有轨道名称和折叠/展开按钮
   - 轨道内的元素可拖拽调整位置

### 视觉风格要求：

1. **配色方案：**
   - 主背景色：深灰色 (#2B2B2B)
   - 次要背景色：稍浅灰色 (#3C3C3C)
   - 工具栏/面板背景：#353535
   - 主要文字：白色 (#FFFFFF)
   - 次要文字：浅灰色 (#CCCCCC)
   - 强调色/按钮：蓝色 (#1890FF) 或品牌色
   - 成功/确认：绿色 (#52C41A)
   - 警告/删除：红色 (#FF4D4F)

2. **字体：**
   - 界面字体：微软雅黑或系统默认无衬线字体
   - 字号：主要文字14px，标题16-18px，小字12px

3. **图标：**
   - 使用现代扁平化图标
   - 工具栏按钮图标 + 文字标签
   - 播放控制使用标准播放/暂停等图标

4. **间距与对齐：**
   - 组件间距：8px, 16px, 24px
   - 面板内边距：16px
   - 卡片间距：12px

5. **交互状态：**
   - 按钮hover：亮度增加10%
   - 按钮active：稍微缩小
   - 选中状态：蓝色边框或背景高亮

### 重点组件设计：

1. **时间轴编辑器**
   - 这是最复杂的组件，需要特别注意
   - 时间刻度尺要清晰，刻度间距均匀
   - 片段显示为矩形卡片，有阴影和圆角（4px）
   - 拖拽手柄在片段边缘
   - 播放指针要醒目（红色，2px宽）

2. **视频播放器**
   - 16:9或9:16比例的黑色播放区域
   - 中央有播放图标（未播放时）
   - 播放控制条设计要清晰易用

3. **素材卡片**
   - 缩略图占据卡片上半部分
   - 下方显示文件名（截断过长文本）
   - 右上角显示时长标签（黑色半透明背景）

4. **AI分析结果列表**
   - 每项有边框，内边距充足
   - ✅和❌图标要醒目
   - 精彩度分数用进度条或数字+颜色表示

### 参考风格：

- 参考剪映桌面版的专业感
- 参考Adobe Premiere Pro的时间轴设计
- 参考DaVinci Resolve的颜色搭配
- 整体要现代、专业、不拥挤

### 设计交付：

请生成包含以下界面状态的设计：
1. 主界面（已导入素材，时间轴有内容）
2. AI分析进度对话框
3. AI分析结果展示状态
4. 导出配置对话框

### 技术要求：

- 设计分辨率：1920x1080（桌面应用标准）
- 最小窗口尺寸：1280x720
- 界面要响应式，支持窗口缩放
```

## 补充说明

- 素材库显示"拖拽或点击导入视频素材"提示

- 时间轴为空

- 素材库显示3-5个视频卡片

## 使用建议

1. **在Pencil中分阶段设计：**
   - 第一阶段：整体布局和主界面
   - 第二阶段：时间轴详细设计
   - 第三阶段：各种对话框和状态

2. **重点关注：**
   - 时间轴的专业感和可用性
   - 整体配色的协调性
   - 组件间距的一致性

3. **可以调整的地方：**
   - 配色方案（可以选择更有品牌特色的颜色）
   - 图标风格（扁平化或线性图标）
   - 组件圆角大小

4. **交付后续：**
   - UI设计确认后，前端团队基于设计稿开发
   - 设计稿可作为开发参考和视觉规范
